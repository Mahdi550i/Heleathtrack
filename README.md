# HealthTrack

## Project Description
HealthTrack is a simple wellness tracking web application built using **ReactJS** and **Tailwind CSS**.  
The project includes multiple pages such as Home, About, Features, Contact, and a dynamic Dashboard page.  
Users can log their daily activities, meals, and workouts, and the data is stored in the browser using **localStorage**.  
The application is fully responsive and works on both desktop and mobile devices.
